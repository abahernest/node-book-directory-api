# Show A Book

Show a specified book

**URL** : `/books/:id`

**Method** : `GET`

**Auth required** : YES

**Data constraints** : NO

## Success Responses

**Code** : `200 OK`

**Content** : `{}`

**Example** : 

```json
{
    "_id": "602dd2eba27104130ce6042f",
    "title": "Book of Moses",
    "author": "Steven Jobs"
}
```
